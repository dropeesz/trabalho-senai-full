## 📌 Sobre o Projeto

Este é o **Trabalho Final desenvolvido no SENAI**, que reúne os conhecimentos adquiridos ao longo do curso técnico em programação/frontend.  
O objetivo do projeto é aplicar técnicas de **HTML, CSS e JavaScript** para construir uma página web funcional e visualmente organizada.

O projeto demonstra lógica de programação via JavaScript, manipulação de elementos com DOM e layout responsivo com CSS.

---

## 🚀 Funcionalidades

- 🏠 Estrutura de layout organizada com HTML semântico  
- 🎨 Estilização visual com CSS  
- 🔄 Interatividade com JavaScript  
- 📷 Uso de imagens e galerias  
- 📍 Navegação fluida entre seções

## 📌 Como Visualizar o Projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/dropeesz/trabalho-senai-full.git
   ```
2. Abra o arquivo `index.html` no navegador:
   - Basta dar **duplo clique** ou
   - Arrastar e soltar no navegador do seu computador

> 💡 Não é necessário servidor backend para rodar — funciona diretamente no navegador.
